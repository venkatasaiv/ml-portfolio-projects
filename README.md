# ML Portfolio Projects

**A collection of machine learning projects covering computer vision, natural language processing, and recommendation systems — built using Python, TensorFlow, PyTorch, and Scikit-learn.**

---

## Projects

### 📸 Image Classification — `image-classification/`

CNN and transfer learning models for image recognition tasks.

- **Datasets**: CIFAR-10, Fashion-MNIST, MNIST
- **Techniques**: Convolutional Neural Networks, Transfer Learning, Data Augmentation
- **Frameworks**: TensorFlow/Keras, PyTorch

### 💬 Sentiment Analysis — `sentiment-analysis/`

NLP pipeline for classifying text sentiment from product and social media reviews.

- **Datasets**: IMDB Reviews, Amazon Reviews
- **Techniques**: BERT, LSTM, TF-IDF, Word2Vec
- **Libraries**: HuggingFace Transformers, NLTK, Scikit-learn

### 🎯 Recommendation System — `recommendation-system/`

Collaborative and content-based filtering for personalized recommendations.

- **Datasets**: MovieLens, Amazon Product Data
- **Techniques**: Matrix Factorization, Collaborative Filtering
- **Libraries**: Scikit-learn, Pandas, NumPy

---

## Tech Stack

- **Languages**: Python 3.8+
- **ML Frameworks**: TensorFlow, PyTorch, Scikit-learn, Keras
- **Data**: Pandas, NumPy, Matplotlib, Seaborn
- **Environment**: Jupyter Notebook, VS Code

---

## Project Structure

Each sub-project follows this layout:

```
project-folder/
├── README.md           # Project-specific documentation
├── data/               # Data loading and preprocessing
├── src/                # Model training and evaluation code
├── notebooks/          # Exploratory analysis and experiments
├── results/            # Metrics, plots, and outputs
└── requirements.txt
```

---

## Setup

```bash
git clone https://github.com/venkatasaiv/ml-portfolio-projects.git
cd ml-portfolio-projects

pip install -r requirements.txt
```

Then open the relevant notebook in `notebooks/` for whichever project you want to explore.
